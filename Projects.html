Project 1: Flutter Fitness Tracker App

The Flutter Fitness Tracker is a comprehensive mobile application that enables users to dynamically log and manage their workouts and exercises. The app uses the Hive database to store user data, such as names, workouts, exercise details, weights, and consistency progress.

Key Features

  • Dynamic Workout Management: Users can log new workouts and exercises, specifying details like weights, repetitions, and sets.
  • Data Persistence: Utilizes Hive database for persistent storage of user data, ensuring that workout history and progress are maintained across sessions.
  • Progress Visualization: Includes a heatmap widget to visualize monthly workout consistency, helping users track their fitness goals.

Technologies Used

  • Languages: Dart
  • Framework: Flutter
  • Database: Hive
  • Tools: Android Studio, Visual Studio Code

Project 2: Unity C# Assessment App
